## Что это такое

Автоматический Excel-отчёт, который показывает, **сколько мы должны клиентам** — то есть товары, за которые клиенты заплатили, но ещё не получили (зарезервированные устройства).

**Файл:** `0_Inbox/Задолженность_перед_клиентами_v2.xlsx`  
**Google Таблица (обновляется автоматически):** https://docs.google.com/spreadsheets/d/1GuFU4MaFG9KagIxVbeUW6QV2RuLtj_OebARltjoMrz0/edit

## Листы отчёта

| Лист | Назначение |
|---|---|
| **Сводка** | Дашборд: ключевые цифры, разбивка по категориям, ТОП-10 должников |
| **Бризеры** | Детализация по брендам и моделям бризеров (AIRNANNY, Tion, Ballu…) |
| **Товары (все)** | Все товары в резерве с фильтрацией по категориям |
| **Детализация** | Построчная привязка: клиент → заказ → товар → долг → себестоимость |
| **Закрытые с долгом** | Аномалии: закрытые заказы с ненулевым долгом (требуют разбора) |
| `_API_Клиенты` | Исходные данные по клиентам (из МойСклад) |
| `_API_Позиции` | Исходные данные по позициям заказов |
| `_Справочник` | Закупочные цены товаров (можно корректировать вручную) |

## Логика расчётов

### Кто попадает в отчёт

- Контрагенты с **положительным балансом** (мы должны им деньги/товар)
- Заказы с **2023-01-01** по сегодня
- Исключены: Микроклиматика, ИП Гончаров, ООО Бризекс, тестовые контрагенты

### Долг клиента

```
Долг = payedSum - shippedSum  (из заказа покупателя)
```

Распределяется по позициям **пропорционально дисконтированной цене** каждой позиции.

### Себестоимость

Берётся `buyPrice` из карточки товара в МойСклад. Если не указана — `kitPrice` (закупочная/минимальная цена из прайс-листа). В листе `_Справочник` можно скорректировать вручную.

### Категории товаров

| Категория | Признак |
|---|---|
| **Бризер** | Слова: airnanny, tion, бризер, fanmaster, ballu, royal clima, turkov, и др. в названии/папке |
| **Сплит/Кондиционер** | Слова: сплит, кондиционер, split, inverter |
| **Услуга** | `meta.type = "service"` |
| **Прочее** | Фильтры, запчасти, аксессуары |

## Быстрый старт

### Обновить отчёт вручную (через Cursor AI)

Напишите в чат:
> "Обнови отчёт по задолженности перед клиентами"

ИИ-агент выполнит два шага:
1. Загрузит свежие данные из МойСклад API
2. Сформирует новый xlsx-файл

### Запустить скрипт напрямую

```bash
cd "0_BREEZEKS/3. РЕСУРСЫ/391-БИБЛИОТЕКА/База знаний по МойСклад/Отчёт — Задолженность перед клиентами/scripts"
python3 01_fetch_data.py
python3 02_build_report.py
```
